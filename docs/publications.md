# How to cite

If you use the catalogue or its Python API in research, software, teaching, or operational work,
please cite the relevant publication and link back to the repository.

## Bibtex

Use the following bibtex entry:

```bibtex
@misc{montero2026towards,
  doi = {10.48550/ARXIV.2606.13923},
  url = {https://arxiv.org/abs/2606.13923},
  author = {Montero,  David and Aybar,  César and Mahecha,  Miguel D. and Gómez-Chova,  Luis},
  keywords = {Databases (cs.DB),  Digital Libraries (cs.DL),  FOS: Computer and information sciences,  FOS: Computer and information sciences},
  title = {Towards an open registry of Earth observation instruments},
  publisher = {arXiv},
  year = {2026},
  copyright = {Creative Commons Attribution 4.0 International}
}
```

## APA

Use the following APA citation style:

```
Montero, D., Aybar, C., Mahecha, M. D., & Gómez-Chova, L. (2026). Towards an open registry of Earth observation instruments. https://doi.org/10.48550/ARXIV.2606.13923
```
